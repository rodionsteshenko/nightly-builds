# Nightly Builds

Small CLI tools built by Coby, one per night.

Each project is self-contained, zero-dependency Python that solves a specific problem.

## Projects

| Project | Description |
|---------|-------------|
| **quote-bank** | Personal quote collector. Add quotes with source/speaker/tags, get random quotes, search, export. |
| **mcu-tracker** | Track your MCU watching progress. Release order, timeline order, post-credits reminders. |
| **anime-tracker** | Track anime watching. Fetches episode counts from AniList API. |
| **commit-msg** | AI-assisted git commit messages using Claude CLI. |
| **setlist-builder** | Concert setlist generator with pacing and transition scoring. |
| **lyric-deep-dive** | Pop-Up Video style song analysis. Fetches lyrics, finds patterns. |
| **terminal-dashboard** | Quick terminal status dashboard. |
| **playlist-to-blog** | Convert Spotify playlists to blog posts. |
| **spotify-auth** | Spotify OAuth helper. |

## Philosophy

- One evening's work
- Zero external dependencies (stdlib only)
- Actually useful
- 100-500 lines
